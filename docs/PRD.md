# BrightRoots - Product Requirements Document (PRD)

## 1. Executive Summary

BrightRoots is a Parenting Decision-Support and Child Development Platform designed to help parents make informed decisions regarding their children's growth and wellbeing. The platform focuses on promoting healthy habits through tracking, personalized recommendations, educational resources, and actionable guidance.

Unlike traditional awareness platforms that primarily highlight problems, BrightRoots emphasizes positive habit formation, family engagement, and practical solutions. The platform aims to help parents build healthier routines related to digital wellbeing, sleep, outdoor activities, and overall child development.

---

# 2. Problem Statement

Modern parents face numerous challenges in supporting healthy childhood development.

Common issues include:

- Excessive screen time exposure.
- Reduced outdoor activity.
- Poor sleep habits.
- Limited family interaction.
- Difficulty identifying age-appropriate alternatives to digital entertainment.
- Information overload from multiple online sources.

Although parenting advice is widely available through articles, videos, and social media, parents often struggle to convert information into consistent action.

There is a need for a platform that helps parents:

- Track important habits.
- Understand current behavior patterns.
- Receive personalized recommendations.
- Build sustainable healthy routines.

# 5. Functional Requirements

## FR-1 User Authentication

The system shall provide:

- User Registration
- User Login
- Secure Authentication
- Protected User Sessions

---

## FR-2 Parent Dashboard

The system shall provide a dashboard displaying:

- Child overview
- Habit summaries
- Progress indicators
- Recommendations

---

## FR-3 Child Profile Management

Parents shall be able to:

- Create child profiles
- View child profiles
- Update child information

Profile information includes:

- Name
- Age
- Gender

---

## FR-4 Screen Time Tracking

Parents shall be able to record:

- Daily screen time usage
- Device usage patterns

The system shall:

- Store screen-time records
- Display trends and summaries

---

## FR-5 Sleep Tracking

Parents shall be able to record:

- Daily sleep duration

The system shall:

- Monitor sleep patterns
- Display sleep statistics

---

## FR-6 Outdoor Activity Tracking

Parents shall be able to record:

- Outdoor play duration
- Physical activities

The system shall:

- Track activity trends
- Display activity summaries

---

## FR-7 Recommendation Engine

The system shall generate recommendations based on:

- Screen time habits
- Sleep habits
- Outdoor activity levels

Recommendations should be:

- Practical
- Actionable
- Easy to understand

---

## FR-8 Parenting Resource Library

The system shall provide educational content related to:

- Digital wellbeing
- Healthy routines
- Family interaction
- Child development

# 7. MVP Scope

The first version of BrightRoots will include:

## Included

- Authentication
- Parent Dashboard
- Child Profile Management
- Screen Time Tracking
- Sleep Tracking
- Outdoor Activity Tracking
- Recommendation Engine
- Parenting Resource Library

## Excluded

- AI Parenting Assistant
- Community Forums
- Voice Assistance
- Regional Language Support
- Advanced Analytics
- Mobile Application

These features are reserved for future releases.

---

# 8. Success Metrics

The MVP will be considered successful if:

- Parents can create accounts successfully.
- Parents can manage child profiles.
- Habit tracking data can be stored and retrieved.
- Personalized recommendations are generated.
- Dashboard statistics are displayed correctly.
- Resource content is accessible and organized.
- Users can complete primary workflows without assistance.

# 10. Project Philosophy

Every feature added to BrightRoots must earn its place.

The platform should:

- Focus on actionable guidance.
- Encourage positive habit formation.
- Support healthy childhood development.
- Avoid fear-based messaging.
- Help parents implement improvements consistently.

BrightRoots is not merely an awareness platform. It is a decision-support system designed to help parents turn knowledge into meaningful action.
